Graduation Project

Electronic forms archive

This system aims to facilitate the management of electronic forms for educational departments through a unified platform that gathers all department forms in one place. The system allows users to browse forms easily without the need to log in, while only registered employees are granted the ability to upload forms related to their respective departments, with the add option hidden from other users. The project features a flexible user interface that displays administrative departments in a visually organized manner, reflecting each department's identity through custom colors and design elements, which enhances the user experience and simplifies interaction with the system.

Technologies Used : HTML - CSS - JavaScript - MongoDB - Express
